CONESV - 2º Congresso Nacional de Emergências e Segurança Viária

Bem-vindo ao repositório do projeto web do CONESV. Este projeto contém as páginas oficiais para o 2º Congresso Nacional de Emergências e Segurança Viária, um evento de grande escala que reúne especialistas, pesquisadores e profissionais da área. O evento engloba também o Desafio Nacional de Salvamento Veicular e Trauma e o treinamento Holmatro Rescue Experience.

📁 Estrutura do Repositório

O projeto é composto pelos seguintes ficheiros principais:

index.html: Landing page principal do congresso. Contém informações detalhadas sobre o evento, programação, palestras, local (Parque Novo Mato Grosso, Cuiabá - MT), patrocinadores e links para inscrições.

desafios.html: Página dedicada ao Desafio Nacional de Salvamento Veicular e Trauma. Detalha o funcionamento da competição, categorias (Salvamento Veicular e Atendimento ao Trauma), quem pode participar e o cronograma (timetable) das equipas.

rescue-experience.html: Página com detalhes sobre a imersão prática e treinamento do Holmatro Rescue Experience.

🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando tecnologias front-end modernas, focadas em desempenho e responsividade:

HTML5

Tailwind CSS (via CDN para uma estilização rápida, responsiva e moderna)

JavaScript (Vanilla) (para interações de UI, como menus mobile, navegação por abas, modais/pop-ups e animações de interface)

AOS (Animate On Scroll) (biblioteca para animações fluídas ao fazer scroll na página)

Font Awesome (para ícones vetoriais em toda a interface)

🚀 Como Executar Localmente

Como o projeto é composto inteiramente por ficheiros estáticos (HTML, CSS e JS executados do lado do cliente), não é necessária nenhuma configuração de ambiente complexa.

Faça o clone deste repositório ou descarregue os ficheiros em formato ZIP.

Navegue até à pasta do projeto.

Abra qualquer ficheiro .html (index.html, desafios.html, etc.) diretamente no seu navegador de preferência.

(Dica: O uso de um servidor local, como a extensão Live Server do VS Code, é recomendado para testar o site simulando um ambiente real).

🌐 Hospedagem (Deploy)

A infraestrutura e o domínio deste projeto estão hospedados através da Hostinger. Os arquivos estáticos estão configurados para serem servidos diretamente através da plataforma de hospedagem da Hostinger.
